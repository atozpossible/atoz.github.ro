HOW TO CONVERT OCTAL TO DECIMAL?

An octal number refers to the base-8 numbering system. The octal numbering system uses the numerals 0-1-2-3-4-5-6-7. In computing environments, it is commonly used as a shorter representation of binary numbers by grouping binary digits into threes. The chmod command in Linux or UNIX uses octal to assign file permissions.


The decimal number uses digits from 0 to 9 to represent a number with base 10. The number is expressed in base-10 where each value is denoted by 0 or first nine positive integers.
