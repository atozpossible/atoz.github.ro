HOW TO CONVERT BINARY TO OCTAL?

Binaryis the simplest kind of number system that uses only two digits of 0 and 1 (i.e. value of base 2). Since digital electronics have only these two states (either 0 or 1), so binary number is most preferred in modern computer engineer, networking and communication specialists, and other professionals.

But Octal number refers to the base-8 numbering system. The octal numbering system uses the numerals 0-1-2-3-4-5-6-7. In computing environments, it is commonly used as a shorter representation of binary numbers by grouping binary digits into threes. The chmod command in Linux or UNIX uses octal to assign file permissions.

Example: Convert this binary number into a decimal number
=(1010111100)2
STEPS:
Therefore, Binary to octal is (1010111100)2
1. Making a group of three:
(1010111100)2= 1 010 111 100
2. Adding two 0 to complete the set
= 001 010 111 100
Then write the octal equivalent number:
001=1, 010=2, 111=7, 100=4
Answer: (1010111100)2= (1274)8
