HOW TO CONVERT OCTAL TO BINARY?

Octal number has only 8 symbols: 0, 1, 2, 3, 4, 5, 6, and 7.

But the binary number is the simplest kind of number system that uses only two digits of 0 and 1 (i.e. value of base 2). Since digital electronics have only these two states (either 0 or 1), so binary number is most preferred in modern computer engineer, networking and communication specialists, and other professionals.

Example: Convert this an octal number into binary number
=(2567)8
The conversion can be done by finding the binary equivalent for each digit of the octal number, combining them together in the same order.

STEPS:
1. Separate the digits of the given octal number, if it contains more than 1 digit.
Our octal number is 2567 (separation of each digit 2, 5, 6, 7)
2. Find the equivalent binary number for each digit of octal number. Add as many 0's to the left, if any of the binary equivalent is shorter than 3 bits.
2= 010, 5=101, 6=110, 7=111
3. Write the all group's binary numbers together, maintaining the same group order provides the equivalent binary for the given octal number.
Answer: (2567)8= (10101111)2 [(010101111)2]
